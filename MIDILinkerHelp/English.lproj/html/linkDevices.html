Linking Endpoints

Linking

To link a source endpoint to a destination endpoint:

  1. Select the source endpoint in the 'source' list (upper panel)
  2. Select the destination endpoint in the 'destination' list (lower panel)
  3. Click the 'link' button

If everything works, the link should appear in the links list on the right-hand side of MIDILinker. Your devices are now linked, and MIDI commands sdhould flow from the source to the destination. When a command is sent, ther 'activity' column in the links table will flash on.

Unlinking

To remove a link:

  1. Select the link in the right-hand panel
  2. Click the 'unlink' button below the panel

The link will vanish from the links table and the devices will be unlinked.
